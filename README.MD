## slm - System Load Manager

A very simple console application for monitoring system load.
Written in RUST using the RATATUI framework and a small sysinfo library for obtaining system information.

-----------------------------
Dependencies:

 - sysinfo="0.32.0"
 - chrono = "0.4.38"
 - crossterm = "0.28.1"
 - ratatui = "0.29.0"
